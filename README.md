# react-chat-app
## About 
This is a ___real time___ MERN stack chat application, in which users can create their own account and can add friends to communicate with each other
## Technologies used
* React
* React bootstrap
* Redux
* Node.js
* Express
* MongoDB
* Mongoose
* Socket.io
* Bcryptjs
* Jsonwebtoken
